collective.audiovisualmaterial Installation
---------------------------------

To install collective.audiovisualmaterial using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``collective.audiovisualmaterial`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        collective.audiovisualmaterial
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
